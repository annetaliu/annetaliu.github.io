如何快速地创建自己的博客？ -Jekyll on github 是一个灰常棒的选择。

#从域名到服务器都是免费的： 
	- geithub上创建项目，上传jekyll的代码即可。
	
#更新管理内容方便： 
	- markdown等标记文档把格式和内容都掌控在自己手里。

#丰富多样的主题模版提供选择：
	- 自己创建或者去rubygem网站找模版。
